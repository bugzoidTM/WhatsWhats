# 🎬 Roteiro para Gravação do Vídeo

**Objetivo:** Ensinar leigos a instalar o Agente de IA + Chatbot WhatsApp em poucos passos.

---

## Comparação: Antes vs Agora

| Antes (YouTube) | Agora (Este projeto) |
|-----------------|------------------------|
| 4 programas (VS Code, Git, Node, arquivos) | **Apenas Node.js** |
| Abrir VS Code, terminal, npm install, node chatbot.js | **2 cliques no INICIAR.bat** |
| QR Code no terminal (difícil de ver) | **QR Code no navegador** |

---

## Roteiro sugerido para o vídeo (5–7 min)

### 1. Introdução (30 s)
- "Chatbot WhatsApp que você instala em 3 passos. Sem mensalidade."
- "O bot já responde 'oi' na hora. Depois você adapta para sua empresa."

### 2. Passo 1: Instalar Node.js (1 min)
- nodejs.org → botão LTS → Instalar
- "Único programa que precisa instalar."

### 3. Passo 2: Baixar, descompactar e iniciar (1 min)
- "Baixe o zip, descompacte a pasta."
- Dois cliques no INICIAR.bat
- "Primeira vez: espera 1–2 min. Copie o link localhost:3000 e cole no navegador."

### 4. Passo 3: Conectar e testar (1 min)
- QR Code aparece na tela → escanear com WhatsApp
- "Conectou? Mande 'oi' de outro celular — já responde!"
- Mostrar a resposta automática

### 5. Configurar IA (opcional, 1 min)
- "Quer IA para outras perguntas? Aba Chave Groq."
- Clicar no botão "Obter chave GRÁTIS no Groq" → abre o site
- Criar conta, gerar API Key, copiar, colar, Salvar
- "Pronto. Agora perguntas como 'qual horário?' a IA responde."

### 6. Personalizar (opcional, 30 s)
- "Aba Fluxos: edite as mensagens com nome da sua empresa."
- "Aba Modelo e Prompt: coloque o jeito que a IA deve falar."

### 7. Encerramento (20 s)
- "Mantenha a janela aberta e o PC ligado."
- "Passo a passo completo no PASSO_A_PASSO.md."
